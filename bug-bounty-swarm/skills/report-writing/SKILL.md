# Report Writing Skill

## Title Formula
`[VULN_TYPE] [IMPACT] in [COMPONENT]`

## HackerOne template
```markdown
# Summary

# Steps to Reproduce
1.
2.

# Impact

# Supporting Material/References
```

## Bugcrowd template
```markdown
## Vulnerability Details

## Reproduction Steps

## Expected vs Actual Results

## Impact
```

## Intigriti template
```markdown
## Description

## Proof of Concept

## Business Impact

## Remediation
```

## Impact statement library
- **Confidentiality:** Unauthorized access to sensitive data.
- **Integrity:** Unauthorized modification of protected records or workflows.
- **Availability:** Service degradation or misuse causing user-impacting downtime.
